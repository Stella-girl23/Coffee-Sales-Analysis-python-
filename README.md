# Coffee Sales Analysis ☕

Project Overview
This project analyzes coffee sales data to understand customer behavior, product performance, and revenue trends.

Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

Key Insights
- Latte coffee products generate the highest revenue
- Peak sales occur during 10th hour of the day
- Customers show preference for Card payment methods

Recommendations
- Increase stock of Latte Coffee
- Optimize staffing during the peak hour
- Improve card payment system efficiency

Files
- Coffe_sales.csv → Dataset
- coffee_analysis.ipynb → Analysis notebook
